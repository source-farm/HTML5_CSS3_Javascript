
================================================================================
                              Общая информация
================================================================================

 В этом уроке рассматриваются основы создания HTML документов: базовые HTML
элементы, общая структура документа, создание ссылок и т.д. В основном вся
теория находится в файле ./tutorial/start.html. В других html файлах возможно
идёт объяснение работы каких-то специфичных элементов.
